# mini-shopping-app

## 1. 개요
바닐라 자바스크립트(Vanila JS)를 사용하여 간단한 필터기능 구현.
Tshirt, Pants, Skirt, Blue, Yellow, Pink 버튼을 눌러 원하는 옷만 모아 볼 수 있음.

## 2. 스택
- HTML
- CSS
- JavaScript

## 3. 걸린시간
- 2 days
